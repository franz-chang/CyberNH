# Agent 打猎系统模拟

这个目录是一个独立的小页面，不依赖 `CyberNH` 当前项目代码。

## 文件

- `index.html`: 页面入口
- `styles.css`: 视觉样式
- `app.js`: 数学模型、蒙特卡洛模拟和折线图渲染

## 模型来源

按 PDF 中给出的三条公式实现：

- `C_team = N * C_h`
- `P_success = (N * C_h) / (N * C_h + C_p)`
- `E[M] = P_success * (M_total / N)`

页面同时展示：

- 小型 / 中型 / 大型猎物的理论曲线与模拟曲线
- 按 `60% / 30% / 10%` 刷新概率混合后的总体环境曲线
- 单个 Agent 期望出肉量、团队总期望出肉量、成功率三张折线图
